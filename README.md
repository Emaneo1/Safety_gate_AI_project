# Safety Gate Automotive Recall Risk Model

This project uses machine learning to analyze automotive complaint texts and detect potential safety risks.

The notebook in this repository contains the full analysis and trained models.

## Datasets

The datasets are not included in this repository because they are very large.

To run the notebook, download the datasets from the following sources:

NHTSA complaints dataset  
https://www.nhtsa.gov/nhtsa-datasets-and-apis

Safety Gate (EU RAPEX) dataset  
https://ec.europa.eu/safety-gate

After downloading, save the files in the same folder as the notebook with these names:

complaints.csv  
safetygate_motor_vehicles.csv

## Running the notebook

Open the notebook in Google Colab or Jupyter and run all cells.
